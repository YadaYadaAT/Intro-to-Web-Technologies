Steps to enable LiveServer.bat (Skip if you have already done them in previous meeting)

Install node.js:
https://nodejs.org/dist/v20.10.0/node-v20.10.0-x64.msi

Then open cmd and copy paste:
npm install -g browser-sync

Done...you needed this done only once . From now on you just click on LiveServer.bat and you're good
to go.
